# Keyboard Shortcuts

Quick access to all Markify features via keyboard.

---

## Viewing Shortcuts

Press **F1** anywhere in the application to open the shortcuts panel.

---

## File Operations

| Shortcut | Action |
|----------|--------|
| `Ctrl+O` | Open file dialog |
| `Ctrl+S` | Save current conversion |
| `Ctrl+Shift+S` | Save As (choose location) |
| `Ctrl+N` | New conversion (clear) |
| `Ctrl+Q` | Quit application |

---

## Edit Operations

| Shortcut | Action |
|----------|--------|
| `Ctrl+C` | Copy selected text |
| `Ctrl+V` | Paste / Clipboard Mode |
| `Ctrl+A` | Select all text |
| `Ctrl+Z` | Undo (in preview editor) |
| `Ctrl+Y` | Redo (in preview editor) |

---

## Conversion Modes

| Shortcut | Action |
|----------|--------|
| `Ctrl+V` | Clipboard Mode (paste from Word) |
| `Ctrl+W` | Toggle Watch Mode |
| `Ctrl+D` | Open Diff View |
| `Ctrl+R` | Reverse conversion (MD â†’ DOCX) |

---

## View & Navigation

| Shortcut | Action |
|----------|--------|
| `F1` | Show keyboard shortcuts |
| `F5` | Refresh / re-convert |
| `Ctrl+P` | Toggle Preview pane |
| `Ctrl+B` | Open in browser preview |
| `Ctrl+E` | Open in VS Code |

---

## Options & Settings

| Shortcut | Action |
|----------|--------|
| `Ctrl+,` | Open Options dialog |
| `Ctrl+T` | Cycle themes |
| `Ctrl+Shift+P` | Edit patterns (JSON) |

---

## Quick Actions

| Shortcut | Action |
|----------|--------|
| `Ctrl+Enter` | Convert and save |
| `Escape` | Close current dialog |
| `F11` | Toggle fullscreen |

---

## Mouse Shortcuts

| Action | Effect |
|--------|--------|
| **Drag & Drop** | Drop `.docx` files onto window |
| **Double-click recent** | Re-open previous conversion |
| **Right-click preview** | Context menu with options |

---

## Tips

1. **Clipboard Mode**: Just press `Ctrl+V` after copying from Word
2. **Quick Theme Switch**: `Ctrl+T` cycles through all 7 themes
3. **Fast Save**: `Ctrl+Enter` converts and saves in one step
