# Markify

![Markify Logo](../resources/markify_icon.png)

**Markify** is a zero-dependency Python application that converts Microsoft Word documents (`.docx`) to Markdown (`.md`) format with high fidelity.

## ðŸš€ Quick Start

### For End Users
1. Run **`Markify.bat`** (recommended) or **`dist/Markify/Markify.exe`**
2. Click **SELECT FILE TO CONVERT**
3. Choose one or more `.docx` files
4. Done! Your Markdown files are ready.

### For Developers
```bash
git clone https://github.com/swolfe2/Markify.git
cd Markify
python src/markify_app.py
```
